# 🏥 Patient Record Management System

A modern **Patient Record Management System** developed to simplify hospital and clinic operations by digitizing patient records, appointment management, doctor interactions, and prescription management.

This project demonstrates full-stack development using **Spring Boot**, **Java**, **MySQL**, **HTML**, **CSS**, and **JavaScript**.

---

# 🚀 Features

## 👨‍⚕️ Doctor Module
- Secure Doctor Login
- Doctor Dashboard
- View Patient Records
- Add & Update Prescriptions
- Manage Patient Information

## 👤 Patient Module
- Patient Login
- View Personal Medical Records
- View Prescriptions
- Appointment Information

## 🏥 Reception Module
- Register New Patients
- Manage Appointments
- Update Patient Details

## 🔐 Security
- Spring Security Integration
- Authentication Support
- Secure Backend APIs

# 🚀 Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Kanishka-Rajesh/Patient-Record-Management-System.git
```

Move into the project folder.

```bash
cd Patient-Record-Management-System
```

---

## 2️⃣ Backend Setup

Navigate to:

```text
backend/java/patient-record-management
```

Run the application:

```bash
mvn spring-boot:run
```

or

```bash
./mvnw spring-boot:run
```

---

## 3️⃣ Frontend Setup

Open the required frontend page inside the `frontend` directory.

Example:

```
frontend/login-page/index.html
```

Use **Live Server** in VS Code or any web server to launch the frontend.
